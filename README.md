# Solitaire Open — Mac Setup & Compilation Guide

## Prerequisites

Before starting, install these on your Mac:

### 1. Xcode (Required)
- Open App Store → Search "Xcode" → Install
- After install, open Xcode once to accept the license
- Install command line tools: `xcode-select --install`

### 2. Node.js (Required)
- Download from: https://nodejs.org (LTS version)
- Or install via Homebrew: `brew install node`
- Verify: `node --version` (should be 18+)

### 3. CocoaPods (Required for iOS)
```bash
sudo gem install cocoapods
```

## Step-by-Step Setup

### Step 1: Set up the project
```bash
# Navigate to the project folder
cd solitaire-open

# Install dependencies
npm install

# Initialize Capacitor for iOS
npx cap add ios

# Sync web files to native project
npx cap sync ios
```

### Step 2: Open in Xcode
```bash
npx cap open ios
```

This opens the Xcode project automatically.

### Step 3: Configure Signing (FREE — no Developer Account needed for testing)

In Xcode:
1. Click on "App" in the left sidebar (the project root)
2. Select the "App" target
3. Go to "Signing & Capabilities" tab
4. Check "Automatically manage signing"
5. For "Team", select your **personal Apple ID**
   - If not listed: Xcode → Settings → Accounts → Add Apple ID
6. The Bundle Identifier should be: `com.techfirio.solitaireopen`

### Step 4: Connect your iPhone
1. Plug iPhone into Mac via USB/Lightning cable
2. On iPhone: tap "Trust This Computer" when prompted
3. In Xcode, select your iPhone from the device dropdown (top bar)
   - NOT "Any iOS Device" — pick your actual phone

### Step 5: Build and Run
1. Press the ▶ Play button (or Cmd + R)
2. First time: you'll see "Untrusted Developer" on the iPhone
3. On iPhone: Settings → General → VPN & Device Management
4. Tap your Apple ID → Trust
5. Run again from Xcode — the game launches!

## Testing on Device

### What to check:
- [ ] Cards render correctly at different screen sizes
- [ ] Touch drag & drop works smoothly
- [ ] Double-tap sends cards to foundation
- [ ] Stock pile tap/recycle works
- [ ] Dealing animation plays on new game
- [ ] Hint glow animation shows correctly
- [ ] All screens navigate properly (Welcome → Game → Stats → etc)
- [ ] Nickname signup flow works
- [ ] Leaderboard displays mock data
- [ ] Themes and card backs switch correctly
- [ ] Settings toggles work (draw three, left-handed, etc)
- [ ] Win celebration + fireworks play
- [ ] Daily Challenge uses same seed each day
- [ ] Sound effects play (may need phone not on silent)
- [ ] No content cut off by notch/Dynamic Island
- [ ] No lag during drag operations

### Common Issues:

**"Could not launch app" error:**
→ Go to iPhone Settings → General → VPN & Device Management → Trust

**White screen on launch:**
→ Run `npx cap sync ios` again, rebuild in Xcode

**App expires after 7 days (free signing):**
→ Normal with free Apple ID. Just rebuild from Xcode.
→ Paid Developer Account ($99/yr) removes this limit.
